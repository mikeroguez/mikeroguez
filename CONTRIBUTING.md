# Contributing

This is currently a personal public repository. Contributions should be small,
reviewable, and aligned with the public nature of the site.

Use clear branches and commits. Explain any broad change before making it. Do
not add dependencies without a written reason.

Before proposing a change:

- run `npm run validate`;
- check privacy and public safety;
- verify content against evidence;
- avoid mass rewrites without explanation;
- confirm no private information was copied from local context;
- inspect `git diff`.

Content changes must identify the claim, source, source date, current validity,
sensitivity, and publication authorization.
