# Testing

La base usa Vitest, jsdom y Vue Test Utils.

## Cobertura Inicial

- Montaje de `App.vue`.
- Existencia del enlace para saltar al contenido.
- Navegacion principal.
- Ruta sin hash.

## Comandos

- `npm run test:run`.
- `npm run test:coverage`.
- `npm run validate`.

Agregar pruebas cuando cambien rutas, layout, componentes accesibles o reglas de
publicacion.
