# Accessibility

Objetivo: WCAG 2.2 AA.

Revisar:

- teclado;
- foco;
- landmarks;
- jerarquia de encabezados;
- contraste;
- enlaces comprensibles;
- formularios si se agregan;
- movimiento reducido;
- texto alternativo;
- zoom;
- tecnologias de asistencia.

La accesibilidad debe revisarse desde la estructura, no solo al final.
