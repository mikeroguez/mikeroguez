# Information Architecture

Estructura provisional:

- Inicio: orientacion general y estado editorial.
- Sobre mi: biografia publica verificada.
- Trabajo y proyectos: casos, roles y resultados autorizados.
- Investigacion: publicaciones, lineas de trabajo y evidencia verificable.
- Contacto: canales publicos aprobados.

Esta estructura no es irreversible. Debe revisarse cuando existan publicos,
contenido y objetivos confirmados.
