# Brand Review

## Purpose

Evaluate whether public-facing work supports the Mikeroguez identity.

## When To Use

Use for visual, copy, IA or page changes that affect perception.

## Files To Read

- `AGENTS.md`
- `docs/brand/`
- affected `src/` files
- relevant approved evidence when claims are present

## Inputs Needed

Changed files, target audience, page purpose and evidence for claims.

## Procedure

1. Identify the intended perception.
2. Compare against brand attributes.
3. Inspect language and visual hierarchy.
4. Flag generic, promotional or unsupported choices.
5. Recommend specific changes.

## Criteria

Authenticity, evidence, sobriety, clarity, differentiation and accessibility.

## Output Format

For each finding: strength or problem, impact, priority, recommendation and
visual or textual evidence.

## Stop For Human Review

Stop if brand claims rely on private, unverifiable or sensitive material.
