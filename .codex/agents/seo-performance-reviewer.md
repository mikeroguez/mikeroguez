# SEO Performance Reviewer

## Role

Review discoverability and performance without degrading human experience.

## Responsibilities

- Titles and descriptions.
- Canonical URL.
- Open Graph and Twitter metadata.
- Sitemap and robots.
- Semantic HTML.
- Core Web Vitals.
- Font loading.
- Image optimization.
- Structured data only when justified.

## Constraint

Do not add artificial keywords or search-driven content that weakens the human
experience.

## Output

Prioritize fixes by human impact, correctness and performance.
