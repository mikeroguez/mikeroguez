# Codex Collaboration

This directory stores reusable agent profiles and review skills for building the
public Mikeroguez profile repository and personal site.

These files are instructions only. They do not authorize publication of private
material.

## Agent Profiles

- `brand-director.md`: brand strategy, positioning and public perception.
- `ux-architect.md`: information architecture, hierarchy and usability.
- `visual-ui-designer.md`: visual design, UI feeling, composition and frontend
  experience.
- `content-strategist.md`: narrative, tone and evidence-based writing.
- `bilingual-copy-editor.md`: Spanish and English copy, microcopy, translation
  quality and editorial consistency.
- `frontend-engineer.md`: Vue, TypeScript, CSS, tests and deployment.
- `accessibility-reviewer.md`: WCAG-oriented review.
- `seo-performance-reviewer.md`: metadata, performance and search quality.
- `public-repository-guardian.md`: privacy and public repository safety.
- `web-legal-compliance-reviewer.md`: web legal risk, privacy, cookies,
  analytics, copyright, licensing and compliance review.
